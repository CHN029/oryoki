#### features

---

→ See [user manual](https://github.com/thmsbfft/oryoki/blob/master/USER-MANUAL.md) for a more complete run through.

```
* Frameless browsing
* Picture in picture
* Magic screenshots
* Video recording
* Visual filters (invert, grayscale)
* Devtools
* Search dictionary
* Window dimensions helper
```
